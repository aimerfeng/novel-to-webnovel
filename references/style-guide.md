## 1. 三档程度定义

### 轻度润色
- 只做：标点转换、删注音、去翻译腔句式、日式语尾自然化
- 不加网感词，不改叙事节奏，人名背景全保留
- 适合：用户只是"读着别扭"，不想动味道

### 中度网文化（最常用档位）
- 轻度全部内容 +
- 叙述改第一人称吐槽流：短句化、口语化、允许单句成段制造节奏
- 网感词点缀（每千字 2~3 处封顶）
- 人名、地名、文化背景（校服、便当、社团…）保留
- 适合：想要网文口感但保留原作氛围

### 重度本土化
- 中度全部内容 +
- 人名改中文姓名（需先产出映射表让用户确认）
- 背景搬到中国校园/城市，文化细节替换（汉堡排→红烧肉、文化祭→校园文化节）
- 敬语体系转换（"XX同学"→直呼其名/外号，"前辈"→"学长/学姐"）
- 适合：用户明确要求"看不出是日轻"
- ⚠️ 版权作品不做此档全书处理；映射表必须先经用户确认再动工

## 2. 标点与格式（通用）

| �� | 改 | 备注 |
|---|---|---|
| 「…」 | "…" | 对话 |
| 『…』 | '…' | 引用中引用/电话音/书名按语境 |
| （假名注音） | 删除 | くるみ/むつき 等，脚本处理 |
| 段首全角空格 | 删除 | 网文排版段间空行、段首不缩进 |
| ！？连用 | 保留但不加码 | 不要自行加感叹号 |
| ～ | 酌情保留 | 卖萌语气可留，叙述里删 |

省略号统一……，破折号统一——。章节标题行永远原样保留。

## 3. 翻译腔清除清单

高频翻译腔 → 自然中文（叙述部分）：

| 翻译腔 | 自然替换示例 |
|---|---|
| 我不禁扶额 | 我扶额 / 我一个头两个大 |
| 令人叹服/令人无言 | 服了 / 没话说 / 我竟无言以对 |
| 可有可无的存在 | 存在感约等于零 / 小透明 |
| 露出困扰的表情 | 一脸为难 |
| 说出这种话 | 讲这种话 / 这话都说得出口 |
| 也不为过 | 都不夸张 |
| 总觉得 | 莫名 / 总感觉 |
| 明明……却…… | 保留但别每段都用 |
| 虽说是理所当然 | 废话，当然是这样 / 这不明摆着 |
| 所谓的XX | 直接说XX |
| 无法理解 | 搞不懂 / 属实理解不了 |

日式语尾（对话部分，看人物性格酌情）：
- 「…对吧？」→ "…吧？" / "…是吧？"
- 「…呢」结尾 → 多数直接删；软妹角色可留少量
- 「…哦」→ "哦"可留，但别每句都带
- 「诶？」→ "诶？"/"啊？"看角色
- 「真是的」→ "真是的"可留 / "服了"
- 「请多指教」→ 语境化："以后麻烦你了"等

## 4. 网感词使用规范

**白名单**（中度档可用，每千字 2~3 处封顶）：
心累、破防、社死、离谱、绝了、有一说一、好家伙、属实、这波、免疫、
拉满、上头、下头、无语住了、蚌埠住了（口语角色限定）、CPU干烧了（吐槽限定）、
十级XX、焊死、锁死、单方面宣布、就离谱、麻了

**黑名单**（过时/出戏/太抽象，任何档位不用）：
YYDS、绝绝子、xswl、awsl、集美、耗子尾汁、u1s1（写作"有一说一"）、
夺笋、芭比Q、栓Q、我不李姐、鸡你太美 及一切时效性烂梗

**使用原则**：
- 网感词放在**叙述/内心吐槽**里，不放进角色对话（除非角色本来就是话痨嘴碎型）
- 同一个词全书不超过 5 次，同一章不重复
- 情绪高点、剧情严肃段（告白、冲突、催泪）**清零**网感词，让正文自己说话

## 5. 人物口吻一致性

项目规范里必须建人物表，每人一行：名字 + 口吻锚点。例：

- 胡桃（主角/叙述者）：老实人，无奈型冷静吐槽，不暴躁不刻薄
- 睦月：嘴皮子溜，得寸进尺但不讨厌，对话可以更活
- 星罗：表面温柔实则不好惹，拒绝人毫不留情，说话软刀子

改写对话时先看说话人是谁，再决定语气词和句式。**吐槽风格属于叙述者，
不要传染给每个角色。**

## 6. 对照示范库

### 叙述（中度档）
前：
> 我不禁苦笑。虽然每次都是这样，但她这张伶牙俐齿真是令人叹服。想必吵架从没输过，不过这招对我没用。

后：
> 我扶额，心累。
>
> 每次都这样，这张嘴皮子怕不是十级辩手转世，吵架就没输过。
>
> 可惜，这招对我免疫。

### 对话开场（中度档）
前：
> 「胡桃（くるみ），物理课题借我看」
>
> 比起早安，睦月(むつき)更先说的是这句话。

后：
> "胡桃，物理作业借我抄抄。"
>
> 睦月一进教室，招呼都不打，张口就是这句。

### 场景描写（中度档，注意：描写段落降低网感密度）
前：
> 升上高中后，已经过了半年多。大家也都适应了高中生活，连日常对话也渐渐失去了新鲜感。

后：
> 升上高中已经半年多，大家早就适应了，连日常闲聊都透着一股熟稔的无聊。

### 严肃段（任何档：零网感词）
前：
> 然而现在的我，却——正在帮她系上项圈。

后：
> 可现在的我，却——
>
> 正在给她系项圈。
